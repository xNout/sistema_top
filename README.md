# SISTEMA DE TOP KILLS HECHO EN DX By -Rex-

Un sistema simple de TOP asesinatos hecho en DX, es capaz de actualizarse en tiempo real.

Formas de uso
* **Con F5 Abres el panel**
* **Con /mystats miras tus asesinatos**
